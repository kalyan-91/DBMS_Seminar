    SQL Data Types

    Definition: A data type defines the kind of data a column can store, such as text, numbers, or dates. It helps the database know how to store and process the data.

    Type Use For Max Size Example
    CHAR Fixed text 255 Gender CHAR(1)
    VARCHAR2 Variable text 2000 Name VARCHAR2(50)
    NUMBER Numeric 38 digits Age NUMBER(3)
    DATE Date/Time - DOB DATE
    LONG Large text 2GB Bio LONG
    RAW Binary 255 bytes Photo RAW

    Data Type Scenarios

  • CHAR: Country codes (IN, US) - always 2 letters
  • VARCHAR2: Email addresses - vary in length
  • NUMBER: Product prices, quantities, percentages
  • DATE: Order dates, appointment schedules
  • LONG: Product descriptions, article content
  • RAW: Profile pictures, signature images
    • üí° VARCHAR2 saves space!

    Command Action
    CREATE Build new table
    ALTER Modify structure
    TRUNCATE Remove data only
    RENAME Change table name

    TRUNCATE vs DROP

    TRUNCATE

    ‚úÖ Keeps structure

    ‚úÖ Removes data

    ‚úÖ Reusable

    VS

    DROP

    ‚ùå Deletes structure

    ‚ùå Deletes data

    ‚ùå Gone forever

    Avoid These!

    ‚ùå Don't

    • Forget semicolon
    • Use spaces in names
    • Drop without backup
    • Make columns too small

    ‚úÖ Do

    • Always use semicolon;
    • Use_underscores
    • Backup first
    • Allow room to grow
